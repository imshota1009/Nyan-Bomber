# Contributing to Nyan Bomber

Thank you for your interest in contributing! 🐱💣🌸

## How to Contribute

### Reporting Bugs
1. Check the [Issues](https://github.com/imshota1009/Nyan-Bomber/issues) page to see if the bug has already been reported.
2. If not, open a new issue.
3. Provide as much detail as possible — browser, OS, steps to reproduce, and screenshots.

### Suggesting Features
1. Open a new issue to suggest a feature.
2. Describe the feature clearly and explain how it would improve the gameplay experience.

### Submitting Code Changes
1. Fork the repository.
2. Create a new branch: `git checkout -b feature/your-feature-name`
3. Make your changes and test them locally in your browser.
4. Commit with a clear message: `git commit -m "Add: description of change"`
5. Push to your fork: `git push origin feature/your-feature-name`
6. Open a Pull Request against the `main` branch.

## Code Style
- Keep JavaScript code readable and comment on non-obvious logic.
- Ensure any CSS updates maintain the clean, responsive, and cute look of the UI.
- Test your changes across desktop and mobile devices before submitting.

## Questions?
Feel free to open an issue if you need guidance understanding the codebase.
