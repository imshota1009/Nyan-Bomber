# 🎓 学習ガイド：にゃんボンバーの仕組み (Learning Guide)

このドキュメントでは、ゲーム『にゃんボンバー：肉球と種爆弾の庭園大戦争』がどのような仕組みで動いているかを、専門用語をできるだけ使わずにわかりやすく解説します。

---

## 🎨 1. 全体像

このゲームは、大きく分けて**「画面の見た目を作るファイル」**と**「ゲームの動きをコントロールするプログラム」**の2つで構成されています。

### ① 画面の見た目を作るファイル（ブラウザで表示するためのもの）
*   **nyan-bomber.html**：ゲームのすべてのプログラムやデザイン、音を1つのファイルにまとめたものです。これを開くだけで、インターネットに繋がっていなくてもゲームが遊べます。
*   **nyan-bomber/index.html**：開発のベースとなるファイルで、ゲーム画面の骨組みが書かれています。

### ② ゲームの動きをコントロールするプログラム（nyan-bomber/game/ の中身）
ゲームの動きは、役割ごとに細かくファイルを分けて作られています。

| ファイル名 | 役割 |
|:---|:---|
| **core.js** | ゲーム全体の進行（ゲーム開始、一時停止、ゲームオーバーなど）を管理します。 |
| **player.js** | プレイヤー（操作するネコ）の移動や、ダッシュなどの動きをコントロールします。 |
| **bomb.js** | 爆弾が置かれてから爆発し、炎が広がる処理を担当します。 |
| **enemy.js** | 害獣（モグラや他の敵）がネコを追いかける動き（AI）を担当します。 |
| **boss.js** | 最後に現れる「モグラ団長」の強力な攻撃や体力を管理します。 |
| **audio.js** | ゲーム中に流れるBGMや、爆発したときの効果音を管理します。 |
| **ui.js** | 画面に表示されるハート（体力）やタイマー、スコアなどの表示を更新します。 |

---

## 🛠️ 2. 使われている主な仕組み

### 🌐 3Dグラフィックス (Three.js)
このゲームは3D（立体的）に描画されています。これは「Three.js」という仕組みを使い、ブラウザ上でキャラクターや庭園のブロックを立体的に配置・表示しています。

### 🎵 リアルタイムの効果音
爆発の音やダッシュの音などは、事前に用意された音源を再生するだけでなく、プログラムの中でリアルタイムに音波を合成して鳴らしています。これにより、非常に軽快で心地よい音が鳴り響きます。

### 🧭 敵の追いかけルート（AI）
敵キャラクターは、壁や障害物を自動で避けながらネコに向かって最短ルートで歩いてきます。これは、経路探索（進む道を見つける）プログラムを使って実現しています。

## 🚀 今後の勉強・改造のヒント

このゲームを改造してプログラミングを学ぶためのアイデアです：
1.  **新しいネコを追加する**：`player.js` を改造して、スピードが速いネコや、爆弾をたくさん置けるネコを作ってみましょう。
2.  **新しいステージを作る**：`stage.js` のステージデータを書き換えて、壁の配置や敵の数を変えてみましょう。
3.  **爆弾の威力を変える**：`bomb.js` を変更して、爆発の範囲を広げたり、特殊なエフェクトを追加してみましょう。
